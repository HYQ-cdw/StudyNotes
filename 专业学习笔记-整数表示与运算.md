## 一、整数的二进制编码

### 1.1 无符号整数编码

对于一个w位的二进制向量 `[xw-1, xw-2, ..., x0]`，其无符号数值表示为：

$$\vec{x} 表示的数值 = \sum_{i=0}^{w-1} x_i \cdot 2^i$$

**特点**：
- 范围：0 ~ 2^w - 1
- 能表示的最大值：2^w - 1

### 1.2 补码编码

为了表示有符号数，计算机采用补码（Two's Complement）编码：

$$\vec{x} 表示的数值 = -x_{w-1} \cdot 2^{w-1} + \sum_{i=0}^{w-2} x_i \cdot 2^i$$

**特点**：
- 最高位为符号位，为1时表示负数
- 范围：-2^(w-1) ~ 2^(w-1) - 1
- 负数的表示：对该数的绝对值按位取反后加1

### 1.3 补码设计的巧妙之处

补码的设计让**加减法可以统一处理**：

```
3 - 2 = 3 + (-2)
```

硬件只需要一个加法器就能完成加法和减法运算，大大简化了CPU设计。

## 二、位运算

### 2.1 布尔代数基础

| 操作 | 符号 | 说明 |
|------|------|------|
| 与 | & | 两位均为1时结果为1 |
| 或 | \| | 任一位为1时结果为1 |
| 非 | ~ | 按位取反 |
| 异或 | ^ | 两位不同时结果为1 |

### 2.2 常见位运算技巧

**快速判断奇偶**：
```c
// 判断n是否为偶数
if (n & 1 == 0) // 偶数
```

**交换两个数（不使用临时变量）**：
```c
a = a ^ b;
b = a ^ b;  // b = (a^b)^b = a
a = a ^ b;  // a = (a^b)^a = b
```

**获取第i位的值**：
```c
// 获取n的第i位（从0开始计数）
bit = (n >> i) & 1;
```

## 三、整数运算

### 3.1 加法

普通加法：按位相加，进位处理。

**溢出判断**：
- 无符号加法：当且仅当结果小于任一操作数时发生溢出
- 有符号补码加法：正溢出或负溢出

### 3.2 乘法

- 硬件实现：移位+加法
- 性能：`x * K` 可优化为 `x << log2(K)` 当K是2的幂时

### 3.3 除法

- 整数除法向零取整：-14 / 3 = -4
- `x / 2^k` 可优化为 `x >> k`（逻辑右移）

## 四、学习心得

补码的设计让我深刻认识到计算机底层的优雅与精妙。一个看似简单的编码方案，解决了有符号数表示、加减法统一、溢出处理等一系列问题。好的系统设计，就是在约束条件下找到最简洁优雅的解决方案。
